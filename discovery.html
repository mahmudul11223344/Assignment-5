Question-4: What is event bubbling in the DOM?

Event bubbling means when a event is triggered it access the child element,then the parent , along all the way to main body ,document object,this process is called ,event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

1:Creating element:We can create an element like paragraph or div or section using document.createElement().
2:Adding element:Using appendChild() [parent.appendChild(childElement)] we can add a child element in Parent container.
3.Removing a element:Using removeChild() we can remove a element from a parent div.
